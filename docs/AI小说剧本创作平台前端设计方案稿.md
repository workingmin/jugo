# AI小说剧本创作平台前端设计方案稿

核心参考：以蛙蛙写作（新手友好、网文/短剧垂直适配、引导式操作）、笔灵AI（全能简洁、功能聚合、高性价比体验）为核心参考，兼顾两者前端优势——简化操作路径、强化AI辅助功能可视化、区分小说/剧本双场景，同时规避竞品劣势（蛙蛙剧本功能薄弱、笔灵上下文管理可视化不足），突出“双场景适配、长文本友好、AI全流程赋能”的前端体验，适配10万+字长文本创作与剧本专业格式编辑，满足新手快速上手、专业用户高效创作的双重需求。

## 一、设计总则

### 1.1 设计理念

轻量操作，重度赋能——前端布局简洁有序，核心功能一键可达，弱化技术门槛（贴合蛙蛙写作新手友好特性）；同时将AI辅助、格式编辑、长文本管理等核心功能可视化、便捷化，减少创作干扰，聚焦内容本身；兼顾小说创作的流畅性与剧本创作的专业性，实现双场景无缝切换（弥补蛙蛙、笔灵单一场景侧重的短板）。

### 1.2 核心目标

- 新手适配：操作路径≤3步触达核心创作，引导式输入降低入门门槛（参考蛙蛙写作新手引导）；

- 效率提升：AI功能（续写、润色、转换、大纲生成）一键调用，长文本分章管理、上下文实时同步；

- 双场景适配：明确区分小说/剧本编辑模式，剧本模式支持专业格式（场景、角色、台词），小说模式支持长文本优化；

- 体验一致：多端同步（PC/平板/手机），交互逻辑统一，避免用户学习成本（参考蛙蛙多端适配特性）；

- 差异化突出：强化小说↔剧本双向转换可视化、雪花写作法引导、长文本审校提示，填补竞品前端空白。

### 1.3 目标用户

- 核心用户：网文新手、短剧创作者、兼职编剧（追求高效、简易操作，参考蛙蛙写作核心用户）；

- 次要用户：专业编剧、长篇小说作者（需要专业格式、上下文管理，参考笔灵AI全能适配特性）；

- 潜在用户：内容创业者、自媒体人（需要多场景内容生成、快速导出）。

## 二、核心设计规范（统一视觉与交互，贴合商业产品质感）

### 2.1 色彩规范（贴合创作氛围，简洁不刺眼）

- 主色调：墨蓝色（#1E40AF）—— 贴合文字创作的沉稳氛围，区别于竞品的亮色，提升专业感；

- 辅助色：
        

    - 功能色：绿色（#10B981，用于确认、生成、保存按钮）、橙色（#F59E0B，用于提示、引导、AI功能入口）、红色（#EF4444，用于删除、取消按钮）；

    - 中性色：白色（#FFFFFF，编辑区背景）、浅灰（#F3F4F6，页面背景）、中灰（#9CA3AF，辅助文字）、深灰（#1F2937，正文文字）；

- 色彩禁忌：避免高饱和对比色，编辑区背景与文字对比度≥4.5:1，保障长时间创作不疲劳（参考蛙蛙写作低饱和界面设计）。

### 2.2 字体规范（适配小说阅读与剧本编辑）

- 中文主字体：微软雅黑、思源黑体（无衬线字体，清晰易读，适配多设备）；

- 剧本专用字体：Courier New（等宽字体，符合剧本行业规范，角色名、台词对齐整齐）；

- 字体层级：
        

    - 页面标题：18-22px，加粗，深灰（#1F2937）；

    - 功能标题：16px，加粗，主色调（#1E40AF）；

    - 正文文字（小说）：14-16px，常规，深灰（#1F2937），行高1.8；

    - 剧本文字：12-14px，常规，深灰（#1F2937），行高1.6（等宽字体）；

    - 辅助文字：12px，常规，中灰（#9CA3AF）；

### 2.3 组件规范（统一交互，降低学习成本）

- 按钮：圆角4px，hover状态亮度提升10%，点击状态亮度降低10%；核心功能按钮（生成、导出）加大尺寸，置于显眼位置（参考笔灵AI核心功能突出设计）；

- 输入框：圆角4px，边框浅灰（#D1D5DB），focus状态边框主色调（#1E40AF），提示文字中灰（#9CA3AF）；

- 弹窗：圆角8px，阴影柔和（rgba(0,0,0,0.1)），标题加粗16px，内容区14px，底部按钮对齐右侧（参考蛙蛙写作引导弹窗设计）；

- 标签：圆角2px，背景浅灰（#F3F4F6），文字中灰（#9CA3AF），适配题材、风格等标签展示；

- 进度条：高度4px，背景浅灰（#D1D5DB），进度色主色调（#1E40AF），用于AI生成、导出等进度展示（参考蛙蛙写作生成进度设计）。

### 2.4 交互规范（流畅高效，反馈及时）

- 即时反馈：AI生成、保存、导出等操作，显示进度条+文字提示（如“生成中30%”“保存成功”），避免用户重复操作；

- 快捷操作：编辑区选中文字，右键唤起快捷菜单（续写、润色、删除、格式调整），参考笔灵AI右键快捷功能；

- 引导式交互：新手首次使用，弹出分步引导弹窗（3-4步），标注核心功能入口，可跳过、可回看（参考蛙蛙写作新手引导）；

- 无感知同步：创作过程中自动实时保存（每30秒一次），显示“自动保存中”提示，避免内容丢失；

- 场景切换：小说/剧本模式切换时，提示“格式将自动适配，不会丢失内容”，切换后保留编辑进度。

## 三、页面架构设计（参考蛙蛙、笔灵页面逻辑，分5大核心页面）

整体页面遵循“首页→创作页→作品管理页→素材库页→个人中心页”的逻辑，层级清晰，核心功能一键可达，避免冗余页面；同时支持面包屑导航，便于用户回溯操作，参考蛙蛙写作的简洁层级与笔灵AI的功能聚合特性。

### 3.1 页面层级划分

- 一级页面（核心入口）：首页、创作页、作品管理页、素材库页、个人中心页；

- 二级页面（详情/设置）：作品编辑详情页、剧本格式设置页、小说字数设置页、素材详情页、会员中心页、账号设置页；

- 三级页面（辅助功能）：AI功能设置页、导出设置页、雪花写作法引导页、帮助中心页。

### 3.2 核心页面详细设计（重点，贴合双创作场景）

#### 3.2.1 首页（核心入口页，参考蛙蛙写作首页，简洁引导）

核心目标：快速触达创作入口，展示核心功能，引导新手入门，同时提供历史作品快捷访问，适配不同用户需求。

- 顶部区域（固定）：
        

    - 左侧：平台LOGO（搭配slogan，如“AI赋能，笔落成文”）；

    - 中间：导航栏（首页、创作、作品、素材、帮助），hover显示下拉菜单（辅助功能）；

    - 右侧：个人头像（点击进入个人中心）、会员标识（免费/会员，参考笔灵AI会员展示）、消息通知（铃铛图标，提示保存、更新等信息）；

- 核心区域（自适应）：
       

    - 创作入口（核心，占比40%）：两个大尺寸按钮，分别标注“小说创作”“剧本创作”，按钮下方标注核心优势（如“10万+字长文本支持”“专业剧本格式”），点击直接进入创作页并切换对应模式（参考蛙蛙写作双创作入口）；

    - 新手引导（占比20%）：卡片式设计，标题“新手入门”，内容“3步完成创作：输入构想→AI生成→优化导出”，点击“开始引导”弹出分步教程，右侧设置“跳过”按钮；

    - 历史作品（占比20%）：标题“最近创作”，展示最近3个作品（封面、标题、类型、修改时间），点击直接进入编辑页，支持“查看全部”（跳转作品管理页）；

    - 核心功能推荐（占比20%）：卡片式展示3个核心功能——“AI双向转换”“长文本审校”“雪花写作法引导”，每个卡片搭配图标+简短说明，点击进入对应功能介绍页；

- 底部区域（固定）：

    - 左侧：版权信息、隐私政策、用户协议；

    - 中间：帮助中心入口、反馈入口；

    - 右侧：多端下载入口（PC/手机/平板）、官方联系方式；

- 适配优化：移动端首页简化为“创作入口+历史作品”，核心功能入口置顶，避免滚动过多。

#### 3.2.2 创作页（核心功能页，参考蛙蛙编辑区+笔灵AI功能聚合）

核心目标：实现小说/剧本双模式编辑，AI功能一键调用，长文本友好，格式适配专业，兼顾效率与实用性，是平台核心前端页面。

- 顶部功能栏（固定，参考笔灵AI功能栏）：
        

    - 左侧：返回按钮（返回首页/作品管理）、作品标题输入框（可实时修改，默认“未命名作品”）、保存按钮（手动保存，搭配自动保存提示）；

    - 中间：场景切换按钮（小说/剧本，切换时自动适配格式）、AI核心功能按钮（大纲生成、续写、润色、双向转换），每个按钮搭配图标，hover显示功能说明；

    - 右侧：导出按钮（下拉菜单选择格式：PDF/Word/TXT，剧本额外支持Fountain格式）、字数统计（实时显示当前字数/目标字数，长文本显示分章字数）、设置按钮（进入当前作品设置）；

- 左侧设置区（可折叠，宽度240px，参考蛙蛙写作设置区）：


    - 创作模式设置（小说/剧本分别适配）：
                

        - 小说模式：题材标签（下拉选择/手动添加，如都市情感、悬疑推理）、字数设置（目标字数、分章数量、每章字数范围）、风格设置（下拉选择，如写实、甜宠）、叙事视角（第一/第三人称）；

        - 剧本模式：剧本类型（短剧/电影/电视剧）、时长设置（自动匹配台词字数）、格式设置（角色名对齐、台词缩进、场景描述格式）、语速适配（180-220字/分钟）；

    - AI功能设置：AI生成速度（快速/精准）、上下文关联范围（当前章节/全文）、去AI味强度；

    - 辅助功能（可折叠）：雪花写作法引导（点击进入分步引导，贴合之前适配需求）、角色库（快速插入角色名、角色设定）、伏笔管理（添加/查看伏笔，参考开源工具可视化特性）；

    - 折叠按钮：点击可隐藏左侧设置区，扩大编辑区面积（适合长文本编辑）。

- 中间编辑区（核心，自适应宽度，参考蛙蛙写作编辑区）：
        

    - 小说编辑模式：富文本编辑器，支持加粗、斜体、下划线、段落对齐、分段等基础操作；右侧显示“章节导航”（可快速跳转章节、添加章节、删除章节）；编辑区背景白色，文字清晰，支持夜间模式（切换背景为深灰，文字为浅灰）；

    - 剧本编辑模式：专业剧本编辑器，默认等宽字体，自动区分“场景描述”“角色名”“台词”“动作提示”，输入对应内容自动适配格式（如角色名大写、台词缩进）；支持快速插入场景、角色，点击角色名可关联角色设定；

    - 长文本适配：编辑区支持滚动加载（避免一次性加载过多内容卡顿），分章编辑时，自动保存当前章节进度，切换章节不丢失内容；选中长文本段落，可唤起“分块优化”功能；

    - AI反馈提示：AI生成/润色后，在编辑区右侧显示“优化提示”（如“已优化台词口语化，适配语速”），可点击“撤销优化”；

- 右侧预览区（可折叠，宽度300px，参考笔灵AI预览区）：
        

    - 实时预览：与编辑区实时同步，小说模式预览最终阅读效果（贴合移动端/网页端），剧本模式预览专业格式效果（如打印后的排版）；

    - 预览设置：可切换预览模式（小说/剧本）、调整预览字体大小、查看分章预览；

    - 问题提示：实时检测编辑内容中的问题（如逻辑漏洞、角色设定偏差、格式错误），用红色小图标标注，点击可查看问题详情及优化建议（参考开源工具审校功能）；

    - 折叠按钮：点击可隐藏右侧预览区，扩大编辑区面积。

- 底部状态栏（固定）：
        

    - 左侧：自动保存提示（如“已自动保存，上次保存时间：10:30”）；

    - 中间：分章进度（如“第3章/共10章，当前字数：1.2万字/目标1.5万字”）；

    - 右侧：夜间模式切换按钮、全屏编辑按钮。

#### 3.2.3 作品管理页（参考蛙蛙写作作品管理，分类清晰）

核心目标：实现作品的分类管理、快捷编辑、批量操作，方便用户查找、修改、导出作品，适配长文本作品与多类型作品管理。

- 顶部功能栏（固定）：
        

    - 左侧：页面标题“作品管理”，搭配筛选标签（全部/小说/剧本/草稿/已完成）；

    - 中间：搜索框（支持按标题、题材、创建时间搜索）；

    - 右侧：批量操作按钮（删除、导出、分类）、新建作品按钮（快速进入创作页）；

- 核心区域（自适应，卡片式布局）：
        

    - 作品卡片（参考蛙蛙写作卡片设计）：每个卡片包含“作品封面（自动生成/手动上传）、作品标题、作品类型（小说/剧本）、题材标签、字数/时长、创建时间、修改时间、操作按钮（编辑、导出、删除、预览）”；

    - 分类管理：支持按“类型、题材、状态”分类展示，卡片可拖拽排序（自定义展示顺序）；

    - 草稿管理：单独划分“草稿区”，展示未完成作品，标注“未完成”标签，提醒用户继续编辑；

    - 分页/筛选：支持分页（每页12个作品），筛选条件可组合（如“剧本+短剧+已完成”），方便快速查找；

- 空状态处理：无作品时，显示“暂无作品”提示，搭配“新建作品”按钮，引导用户进入创作页（参考蛙蛙写作空状态设计）。

#### 3.2.4 素材库页（参考笔灵AI素材功能，聚合实用素材）

核心目标：提供小说/剧本创作所需的素材（角色、场景、台词、梗料等），支持素材的查找、收藏、复用，提升创作效率，贴合网文/短剧场景。

- 顶部功能栏（固定）：
       

    - 左侧：页面标题“素材库”，搭配筛选标签（角色/场景/台词/梗料/自定义）；

    - 中间：搜索框（支持按关键词、题材搜索素材）；

    - 右侧：上传素材按钮（支持上传自定义素材）、新建素材按钮（手动创建素材，如角色设定、场景描述）；

- 核心区域（自适应，分类展示）：


    - 素材分类：
               

        - 角色素材：展示角色设定模板（含姓名、性格、背景、动机），支持一键复制到创作页角色库；

        - 场景素材：按题材分类（都市、古装、悬疑等），展示场景描述片段，支持一键插入编辑区；

        - 台词素材：按风格分类（甜宠、悬疑、搞笑等），展示经典台词片段，支持修改后复用；

        - 自定义素材：用户上传/创建的素材，按创建时间排序，支持分类标签管理；

    - 素材卡片：每个素材卡片包含“素材标题、素材类型、题材标签、预览片段、操作按钮（复制、收藏、编辑、删除）”；

    - 收藏功能：点击素材卡片的“收藏”按钮，加入“我的收藏”，方便快速查找常用素材；

- 空状态处理：无素材时，显示“暂无素材，点击上传/新建”提示，搭配热门素材推荐（自动推荐适配当前用户常用题材的素材）。

#### 3.2.5 个人中心页（参考蛙蛙、笔灵个人中心，功能聚合）

核心目标：实现账号管理、会员管理、创作数据查看、设置等功能，提升用户粘性，兼顾实用性与便捷性。

- 顶部个人信息区（固定）：
        

    - 左侧：个人头像（可上传/修改）、用户名、会员等级（免费/普通会员/高级会员）；

    - 右侧：创作数据概览（累计创作字数、作品数量、创作时长），点击可查看详细数据；
