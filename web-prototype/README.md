# AI小说剧本创作平台 - 前端原型设计

基于《JUGO前端设计方案稿》创建的交互式前端原型。

## 项目结构

```
web-prototype/
├── index.html              # 首页
├── create.html             # 创作页（核心功能页）
├── works.html              # 作品管理页
├── materials.html          # 素材库页
├── profile.html            # 个人中心页
├── css/
│   ├── common.css          # 通用样式（设计系统）
│   ├── home.css            # 首页样式
│   ├── create.css          # 创作页样式
│   ├── works.css           # 作品管理页样式
│   ├── materials.css       # 素材库页样式
│   └── profile.css         # 个人中心页样式
└── js/
    ├── common.js           # 通用JavaScript（工具函数、数据）
    ├── home.js             # 首页交互
    ├── create.js           # 创作页交互
    ├── works.js            # 作品管理页交互
    ├── materials.js        # 素材库页交互
    └── profile.js          # 个人中心页交互
```

## 核心功能

### 1. 首页 (index.html)
- **创作入口**：小说创作 / 剧本创作双入口
- **新手引导**：分步引导教程
- **最近创作**：展示最近3个作品
- **核心功能推荐**：AI双向转换、长文本审校、雪花写作法

### 2. 创作页 (create.html)
- **双模式编辑**：小说模式 / 剧本模式切换
- **AI功能**：大纲生成、续写、润色、双向转换
- **左侧设置区**：题材、字数、风格、AI设置、辅助功能
- **中间编辑区**：富文本编辑器、章节管理
- **右侧预览区**：实时预览、问题提示
- **自动保存**：每30秒自动保存
- **夜间模式**：护眼模式切换

### 3. 作品管理页 (works.html)
- **作品筛选**：全部/小说/剧本/草稿/已完成
- **搜索功能**：按标题、题材搜索
- **作品操作**：编辑、导出、删除
- **卡片展示**：封面、标题、类型、字数、时间

### 4. 素材库页 (materials.html)
- **素材分类**：角色/场景/台词/梗料
- **素材操作**：复制、编辑、收藏
- **搜索功能**：按关键词搜索素材

### 5. 个人中心页 (profile.html)
- **个人信息**：头像、用户名、会员等级
- **创作数据**：累计字数、作品数量、创作时长
- **账号管理**：修改用户名、邮箱、手机号
- **会员管理**：月卡/年卡套餐展示
- **创作设置**：默认模式、字体大小、自动保存间隔

## 设计规范

### 色彩系统
- **主色调**：墨蓝色 (#1E40AF)
- **功能色**：
  - 成功/确认：绿色 (#10B981)
  - 警告/提示：橙色 (#F59E0B)
  - 危险/删除：红色 (#EF4444)
- **中性色**：
  - 背景：浅灰 (#F3F4F6)
  - 文字：深灰 (#1F2937)
  - 辅助文字：中灰 (#9CA3AF)

### 字体规范
- **中文主字体**：微软雅黑、思源黑体
- **剧本专用字体**：Courier New（等宽字体）
- **字体层级**：
  - 页面标题：20px
  - 功能标题：16px
  - 正文文字：14px
  - 辅助文字：12px

### 组件规范
- **按钮**：圆角4px，hover状态亮度提升
- **输入框**：圆角4px，focus状态边框主色调
- **卡片**：圆角8px，柔和阴影
- **标签**：圆角2px，浅灰背景

## 响应式设计

### PC端 (≥1200px)
- 三栏布局（创作页）
- 所有功能完整展示
- 支持快捷键操作

### 平板端 (768px-1199px)
- 双栏布局
- 设置区与预览区可切换
- 支持手势操作

### 手机端 (≤767px)
- 单栏布局
- 核心功能优先
- 简化操作流程

## 使用说明

### 本地运行
1. 直接在浏览器中打开 `index.html` 文件
2. 无需服务器，纯静态页面
3. 支持所有现代浏览器（Chrome、Firefox、Safari、Edge）

### 功能演示
- **创作流程**：首页 → 选择创作类型 → 编辑内容 → 使用AI功能 → 保存/导出
- **作品管理**：查看作品列表 → 筛选/搜索 → 编辑/导出/删除
- **素材使用**：浏览素材库 → 复制素材 → 应用到创作中

### 交互特性
- **即时反馈**：所有操作都有提示消息
- **进度展示**：AI生成、导出等操作显示进度条
- **自动保存**：创作页每30秒自动保存
- **实时预览**：编辑内容实时同步到预览区

## 技术特点

### 纯前端实现
- 使用原生HTML、CSS、JavaScript
- 无需后端服务器
- 数据存储在内存中（可扩展为localStorage）

### 模块化设计
- 通用样式与页面样式分离
- 通用功能与页面功能分离
- 易于维护和扩展

### 用户体验
- 流畅的动画过渡
- 友好的错误提示
- 直观的操作流程
