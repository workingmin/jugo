# JUGO Frontend

JUGO AI小说剧本创作平台 - 前端工程

## 技术栈

- **框架**: React 19 + TypeScript
- **构建工具**: Vite 7
- **UI组件库**: Ant Design 6
- **路由**: React Router v7
- **状态管理**: Zustand
- **HTTP客户端**: Axios
- **富文本编辑器**: React Quill
- **样式**: Sass
- **日期处理**: Day.js

## 开发指南

### 安装依赖
```bash
npm install
```

### 启动开发服务器
```bash
npm run dev
```

访问 http://localhost:3000

### 构建生产版本
```bash
npm run build
```

## 核心功能

- 认证系统（登录/注册）
- 作品管理（创建/编辑/删除）
- 小说编辑器（富文本编辑、章节管理、AI辅助）
- 剧本编辑器（专业格式、场景管理、角色管理）
- 导出功能（多格式支持）
